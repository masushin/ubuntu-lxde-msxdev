# ubuntu-lxde-msxdev

ubuntu20.04ベースのMSX開発環境です。以下のものが含まれています。

* lxde + 日本語入力 fcitx
* Visual Source Code
* z88dk (z80向け開発ツール)
* openMSX / openMSX debugger
* MAME + c-bios ドライバ
* nMSXtiles (MSX向けパターンエディタ)
* multipaint (レトロPCフォーマット対応のドローツール)
